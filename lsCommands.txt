1.ls -lh (File size we get)
2.ls -l  (File Permissions)
3.ls -F (Directories with / char )
4.ls -r (In reverse order we get)
5.ls -ls
6.ls -R (directories recursively)
7.ls -ltr (list of files and Directories with latest modification)
8.ls -ls
9.ls -lsh (sort by size)
10.ls -li (by inode number)
11.ls -n (will get user id and group id)
